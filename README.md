 ## Requirements
 core:
 - yay AUR
 ```bash
sudo pacman -S git base-devel curl stow linux-headers flatpak
yay -S ttf-cascadia-code-nerd
 ```

 hyprland:
 ```bash
sudo pacman -S hyprland kitty wofi waybar hyprpaper hyprlock swaync hypridle
yay -S wlogout hyprshot nwg-look
 ```

terminal:
```bash
sudo pacman -S fish starship neofetch
```

gtk:
```bash
yay -S nwg-look catppuccin-gtk-theme-mocha
```

waybar:
```bash
sudo pacman -S bc brightnessctl
```
